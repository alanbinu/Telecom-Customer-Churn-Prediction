# 📂 Dataset

This folder contains the dataset used for the **Telecom Customer Churn Prediction** project.

## Overview

The dataset is a **synthetic telecom customer dataset** designed for building and evaluating customer churn prediction models.

## Dataset Statistics

| Attribute | Value |
|-----------|------:|
| Total Records | 31,251 |
| Total Features | 41 |
| Target Variable | Churn_Flag |

## Target Classes

| Value | Description |
|------:|-------------|
| 0 | Customer Stayed |
| 1 | Customer Churned |

## Feature Categories

The dataset includes features from multiple business domains:

- 👤 Customer Demographics
- 📄 Contract Information
- 📞 Service Usage
- 💳 Billing & Payment History
- 📢 Complaint History
- ⭐ Customer Satisfaction
- 📶 Network Performance
- 🎁 Loyalty & Reward Information

## Machine Learning Workflow

The dataset was used for:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Feature Scaling
- Model Training
- Model Evaluation
- Customer Churn Prediction

## Project Purpose

This dataset was created exclusively for educational and portfolio purposes. It demonstrates a complete end to end machine learning workflow for customer churn prediction using Python and Scikit Learn.
