# Dataset

This project uses a synthetic telecom customer dataset for predicting customer churn.

## Dataset Summary

- Records: 31,251
- Features: 41
- Target Variable: Churn_Flag

## Target Classes

| Value | Meaning |
|-------|---------|
| 0 | Customer Stayed |
| 1 | Customer Churned |

## Features Include

- Customer Demographics
- Contract Details
- Service Usage
- Payment Information
- Complaint History
- Satisfaction Score
- Network Performance

The dataset is intended for educational and portfolio purposes.
