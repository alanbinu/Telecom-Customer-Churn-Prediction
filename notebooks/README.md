# Jupyter Notebooks

This folder contains the notebooks used throughout the Telecom Customer Churn Prediction project.

## Notebook Included

### Telecom_Customer_Churn_Prediction.ipynb

This notebook covers the complete machine learning workflow:

- Data Loading
- Data Exploration
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Model Training
- Model Evaluation
- Feature Importance Analysis
- Business Insights

## Machine Learning Models

The following models were evaluated:

- Logistic Regression
- Decision Tree
- K Nearest Neighbors
- Naive Bayes
- Support Vector Machine
- Random Forest

The Random Forest Classifier achieved the best overall performance with an accuracy of **86.77%**.
