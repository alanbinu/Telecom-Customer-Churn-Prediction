# 🚀 Streamlit Application

This folder contains the **Streamlit web application** for the Telecom Customer Churn Prediction project.

The application provides an interactive interface that allows users to enter customer information and instantly predict whether a customer is likely to churn.

---

## ✨ Features

- 📊 Real time customer churn prediction
- 🤖 Random Forest based prediction model
- 🎯 Interactive user interface
- ⚡ Instant prediction results
- 📈 Business friendly output

---

## 📁 Files

| File | Description |
|------|-------------|
| `app.py` | Main Streamlit application |
| `Scaler.pkl` | StandardScaler used for preprocessing (loaded from the models folder) |

---

# ▶️ How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/alanbinu/Telecom-Customer-Churn-Prediction.git
```

### 2️⃣ Navigate to the project directory

```bash
cd Telecom-Customer-Churn-Prediction
```

### 3️⃣ Install the required dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Launch the Streamlit application

Since the Streamlit application is inside the **app** folder, run:

```bash
streamlit run app/app.py
```

### 5️⃣ Open the application

If the browser does not open automatically, visit:

```
http://localhost:8501
```

## 🧠 Workflow

1. User enters customer information.
2. Input data is preprocessed using the trained StandardScaler.
3. The Machine Learning model predicts customer churn.
4. The prediction is displayed with a user friendly interface.
